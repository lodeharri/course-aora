# course-aora

Este repositorio contiene un curso dedicado al desarrollo de aplicaciones móviles utilizando **React Native**. El objetivo principal es enseñar a los desarrolladores cómo construir aplicaciones móviles robustas y funcionales, cubriendo conceptos clave como componentes, estado, navegación, manejo de datos, y mucho más.

## Tecnologías Utilizadas

- **React Native**: Para el desarrollo de la interfaz de usuario y la lógica de la aplicación.
- **Expo**: Para facilitar el proceso de desarrollo, pruebas y despliegue de la aplicación móvil.
- **JavaScript/TypeScript**: Lenguaje principal para la escritura de código.
- **Firebase**: Para servicios de autenticación, almacenamiento y base de datos en la nube.
- **Git / GitHub**: Para el control de versiones y el manejo del repositorio.

## Objetivos del Curso

- Aprender a construir aplicaciones móviles desde cero.
- Entender los fundamentos de React Native y cómo se integran con otras tecnologías.
- Desarrollar habilidades prácticas en el manejo de componentes, estado, navegación y APIs.
- Familiarizarse con herramientas como Expo, Firebase y Git para mejorar la eficiencia en el desarrollo.

## Estructura del Repositorio

- `src/`: Contiene el código fuente de la aplicación.
- `assets/`: Almacena imágenes, iconos y otros recursos visuales.
- `docs/`: Documentación del curso y ejemplos prácticos.
- `README.md`: Este archivo, que describe el propósito y contenido del repositorio.
